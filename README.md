## 🚀 How to Deploy on GitHub Pages

1. **Create a GitHub account** at https://github.com (if you don't have one)
2. **Create a new repository** — click the `+` button → "New repository"
   - Name it: `muribro-cars` (or any name you like)
   - Set it to **Public**
   - Click "Create repository"
3. **Upload all files**:
   - Click "uploading an existing file"
   - Drag and drop all files from this folder
   - Click "Commit changes"
4. **Enable GitHub Pages**:
   - Go to your repository → **Settings** → **Pages**
   - Under "Source", select **main** branch → **/ (root)**
   - Click **Save**
5. Your website will be live at:  
   `https://YOUR-USERNAME.github.io/muribro-cars/`

## 🖼️ Adding Your Logo

1. Name your logo file exactly `logo.png`
2. Place it in the same folder as `index.html`
3. Upload it to your GitHub repository
4. The logo will automatically appear everywhere on the site!

---

## 💡 Tips

- To update Google Maps on `location.html`:
  1. Go to https://maps.google.com
  2. Search your garage address
  3. Click Share → Embed a map → Copy the iframe code
  4. Replace the `<iframe>` in `location.html`

- All pages share the same `style.css` — changing that file updates the look of the whole website.
